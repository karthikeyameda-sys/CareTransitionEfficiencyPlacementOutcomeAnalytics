# Care Transition Efficiency & Placement Outcome Analytics

Business analytics project analyzing the UAC (Unaccompanied Alien Children) Program's care pipeline — from CBP custody through HHS care to sponsor placement — using process-efficiency metrics rather than simple capacity counts.

## Contents
- `UAC_Care_Pipeline_CLEANED project.csv` — cleaned daily operational dataset (2023–2025)
- `Internship PRoject.pbix` — Power BI dashboard with 5 KPIs, pipeline flow visuals, backlog detection, and weekday/weekend analysis
- `Research_Paper.docx` — full EDA, methodology, findings, and recommendations
- `Executive_Summary.docx` — condensed summary for stakeholders

## Key KPIs
- Transfer Efficiency Ratio
- Discharge Effectiveness Index
- Pipeline Throughput
- Backlog Accumulation Rate
- Outcome Stability Score

## Key Finding
While the pipeline has cleared its historical backlog in aggregate, discharge effectiveness has declined roughly 65% from 2023 to 2025, indicating a process bottleneck that capacity monitoring alone would not reveal.
